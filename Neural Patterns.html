About
Neural Patterns is a web toy built to run simple neural cellular automata (NCA) in the browser. These simple NCAs generate each frame by applying a convolution and activation to each pixel:

Controls:
  • Left Click - Draw pixels of value 1.0
  • Right Click - Draw pixels of value 0.0
  • Scroll - Zoom in/out
  • Spacebar - Pause/Play
The code is open source, feel free to expand on it. Powered with vue, webgl, and lots of coffee.
